Acquiring Bootstrap from a Content Delivery Network (CDN)

Acquiring Bootstrap from a CDN provides some benefits as well as some limitations:

  • The CDN delivers a pure CSS file, therefore, there is no opportunity for preprocessing CSS.
  • There may (or may not: you have to test!) be longer load times.
  • There will be no Bootstrap customization opportunities.
  • The Bootstrap library located at the CDN may have been cached by your user, resulting in a faster startup time.
