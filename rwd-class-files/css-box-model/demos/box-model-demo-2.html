How padding and margin are calculated by the Web browser

Relevant CSS code

body { font-family: Arial, Helvetica, sans-serif; font-size: 20px; } figure { width: 100px; }

Instructions

Follow the steps below to see more information about each div element.

  1. Examine the first div using the browser's web development tools.
    • Right-click the top-most div and choose Inspect.
    • The elements tab of the Developer Tools should be active.
    • Examine the box model properties on the graphic in the developer tools.
    • Click on the computed tab of the styles and examine the box model properties: padding and margin.
  2. Repeat the steps above using the second div on the page.
